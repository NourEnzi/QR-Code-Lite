# Educational QR Code Generator (Lite) 🏫📱

## 📌 Overview
A lightweight, student-friendly, and **purely client-side** Progressive Web Application (PWA) designed specifically for educational environments and school laboratories. This streamlined tool was developed to assist teachers and students in quickly generating Quick Response (QR) codes for science experiments, school projects, and educational resources without the clutter of advanced enterprise features.

**Live Demo:** [Try the QR Code Lite Here](https://nourenzi.github.io/QR-Code-Lite/)

---

## ✨ Key Features
* **Educational Focus:** A simplified, distraction-free user interface tailored for students, featuring a custom pink and purple visual theme.
* **Core Data Types:** Focused specifically on essential educational needs: Text/URLs (for sharing study materials) and Wi-Fi networks (for easy lab connectivity).
* **Privacy Centric & Safe:** **100% Client-Side.** No student data, texts, or links ever leave the device. Perfectly safe for school environments and complies with strict educational privacy standards.
* **Offline & PWA Ready:** Fully installable on school lab computers, iPads, and student phones. Supported by a Service Worker to function flawlessly even when the school's internet connection is down.
* **Responsive UI/UX:** A clean, accessible interface with dynamic Dark/Light modes to reduce eye strain during long study sessions.

---

## 🛠️ Tech Stack
* **HTML5 & CSS3:** Semantic structuring and custom styling (Grid/Flexbox) with a dedicated educational theme.
* **Vanilla JavaScript:** Core business logic and DOM manipulation (Lightweight and fast).
* **easyqrcodejs:** Reliable library for static QR code generation.
* **FontAwesome:** For intuitive UI/UX icons.

---

## ⚠️ Privacy Disclaimer
This tool operates as a strict **Client-Side** utility. All data encoding occurs locally within the browser. We do not track, store, or transmit any inputs to external servers, ensuring a 100% safe browsing experience for students.
